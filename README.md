# Fulusi Digital Bank Landing Page

A modern, responsive landing page for **Fulusi**, a digital banking platform designed specifically for Africa. This project features a clean, professional design with smooth animations and mobile-first responsive layout.

## 🎯 Project Overview

Fulusi is a digital banking solution that aims to provide modern financial services to African markets. The landing page showcases the key features and benefits of the platform with a focus on:

- **Accessibility**: Easy-to-use interface with clear navigation
- **Trust**: Security features and regulatory compliance
- **Innovation**: Modern banking solutions for the African market
- **Mobile-first**: Optimized for mobile devices

## 🌟 Features

### Design Features
- **Modern UI/UX**: Clean, professional design with African-inspired color scheme
- **Responsive Design**: Fully responsive across all devices (mobile, tablet, desktop)
- **Smooth Animations**: CSS animations and JavaScript interactions
- **Mobile Menu**: Hamburger menu for mobile navigation
- **Interactive Elements**: Hover effects and smooth transitions

### Sections
1. **Header**: Fixed navigation with logo and menu
2. **Hero Section**: Compelling headline with call-to-action and phone mockup
3. **Features**: Six key features with icons and descriptions
4. **Benefits**: Comparison with traditional banking
5. **Security**: Trust indicators and security features
6. **Call-to-Action**: App download links and ratings
7. **Footer**: Links, social media, and legal information

## 🚀 Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling with CSS Grid and Flexbox
- **JavaScript ES6**: Interactive functionality
- **Font Awesome**: Icons for UI elements
- **Google Fonts**: Inter font family for typography

## 🎭 Customization

### Colors
Modify the CSS custom properties in `styles.css`:
```css
:root {
    --primary-color: #2E8B57;
    --secondary-color: #FF6B35;
    /* ... other colors ... */
}
```

### Content
Update the HTML content in `index.html`:
- Change company name and tagline
- Update feature descriptions
- Modify contact information

### Animations
Adjust animation settings in `script.js`:
- Counter animation duration
- Scroll animation thresholds
- Hover effect parameters
